# Docker 速查卡

> 快速掌握日期：2026/04/08

---

## 一句话定义

> Docker 是一个把「代码」和「运行环境」打包成「容器」的工具，让你的应用在任何地方都能一致运行。

---

## 核心类比

| 概念 | 类比 |
|------|------|
| Docker 容器 | 集装箱 |
| 代码 + 环境 | 集装箱里的货物 |
| 安装了 Docker 的服务器 | 码头/货船 |

---

## 三个核心命令

| 命令 | 作用 | 类比 |
|------|------|------|
| `docker build` | 把代码打包成镜像 | 造集装箱 |
| `docker run` | 用镜像启动容器 | 启用集装箱 |
| `docker ps` | 查看运行中的容器 | 查看有哪些集装箱在运作 |

---

## 最小示例

```bash
# 拉一个现成的镜像（Ubuntu 系统）并运行命令
docker run ubuntu echo "Hello Docker"

# 查看正在运行的容器
docker ps

# 查看所有容器（包括已停止的）
docker ps -a
```

## 常见错误排查

| 问题 | 解决 |
|------|------|
| 容器启动后立即退出 | 检查启动命令是否阻塞，添加 `-d` 后台运行 |
| 端口冲突 | 使用 `-p 8080:80` 映射到其他端口 |
| 权限不足 | Linux 下加 `sudo`，或配置 docker 用户组 |
